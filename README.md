# Weather and Air Quality Analysis System Using Python

A Python-based data analysis project that analyzes weather and air pollution data of different cities using graphs and visualizations.

### Overview

The Weather and Air Quality Analysis System is a data analysis project developed in Python. This project reads weather and pollution data from a CSV dataset and performs data cleaning, analysis, and visualization.

The system helps users understand:

Average temperature of cities
Relationship between temperature and humidity
PM2.5 pollution levels in different cities

The project uses graphs and charts to present information in a clear and understandable way.

### Problem Statement

Weather conditions and air pollution directly affect human health and the environment. It is difficult to analyze large datasets manually and identify pollution trends or climate patterns.

The purpose of this project is to:

Analyze weather and air quality data efficiently
Visualize environmental conditions using graphs
Compare pollution and weather conditions between cities
Provide meaningful insights from environmental datasets
### Dataset

The project uses a CSV dataset:

pakistan_air_quality_final_clean.csv
Dataset Includes:
City names
Temperature
Humidity
PM2.5 pollution levels
Data Processing:
Duplicate records are removed
Missing values are checked
Data is grouped and analyzed city-wise
